# Sri Lankan Background Image

## Instructions

Place a Sri Lankan landscape image in this directory with the filename:
`sri-lankan-background.jpg`

## Recommended Images

Choose a high-quality image (1920x1080 or higher) featuring:
- Sri Lankan tea plantations
- Sigiriya Rock
- Beautiful beaches (e.g., Mirissa, Unawatuna)
- Elephants in their natural habitat
- Ancient temples
- Lush green landscapes

## Image Specifications

- **Filename**: `sri-lankan-background.jpg`
- **Format**: JPG or PNG
- **Resolution**: 1920x1080 minimum (for Retina displays)
- **Aspect Ratio**: 16:9 or similar
- **File Size**: Optimize to under 500KB for web performance
